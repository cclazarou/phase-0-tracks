Electronic Music

I'm finally familiarizing myself with common electronic music tools, which I've avoided for years because I was too cool to use anything but analog instruments. I'm doing so now because:

  • I have to do it for a Halloween coverband.
  • I've aged out of being cool anyway.

Turns out it's pretty fun! It's fun in the following ways:

  1. It's nice having a new palette to work with after getting used to guitars and pianos.
  2. It's interesting observing the similarities between learning electronic music tools and programming for web development.

Here's one of the things I'm playing with (or at least a fake plugin version of it because it's less than $3000 or whatever a real one is): minimoog

So far I've learned that the compositional tendencies that electronic musicians have, usually born of the tools available, can be avoided by simply avoiding them.
